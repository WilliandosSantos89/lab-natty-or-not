# Vídeo: Um Minuto no Futuro das IAs Generativas 🎥

## 📒 Descrição
Este vídeo de 3 segundos apresenta uma visão rápida e envolvente do potencial e da aplicação das IAs Generativas na vida cotidiana.
Ele mostra um personagem em um ceránio futurista.

## 🤖 Tecnologias Utilizadas
- IA Generativa **[ChatGPT](https://chat.openai.com)** criação do prompt.
- IA Generativa **[Pika.art](https://pika.art/)** para sintetização de voz e geração de avatar virtual;

## 🧐 Processo de Criação
Usei o "ChatGPT" para criar o prompt do vídeo. O "Pilka.art" foi essencial para criar um avatar virtual, trazendo uma dimensão futurística ao vídeo.

## 🚀 Resultados
O vídeo é umum personagem em um ambiente no futurísico.

O vídeo encontra-se anexado ao repositório.

## 💭 Reflexão
Este projeto foi uma experiência incrível em criar ambientes fictícios, utilizando as IAs Generativas.
